# Hub Portal
A (work in progess) Discord bot for checking Hypixel profiles easily on Discord.

## Features
* player stats
  * id
  * last log on
  * skills
  * collection
  * minions
  * fairy souls
  * inventory
  * bank
* guild checking
  * setup inside discord
  * requirement checking
  * member pruning
* **and more to come**
